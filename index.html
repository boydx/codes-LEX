Code Enforcement: A weapon of gentrification?

Lexington, KY


Description

Gentrification is a controversial, polarizing and politically divisive topic. In the city of Lexington, KY, concerns over gentrification have resulted in an increased scrutinization of city agencies. Most prominently, residents have expressed concern that code enforcement violations are being applied aggressively, for the benefit of property developers, and thus have been weaponized as a tool for gentrification. While city officials have denied these claims, the complaint based nature of the current code violation system. This map explores patterns of code enforcement violations, with more detailed information on the type of complaint.

A note on data sources and privacy concerns:

The data for the map was obtained via an open data request from the LFUCG Department of Code Enforcement. The original dataset contained all closed code violations related to residential buildings between 2012 and 2017.
